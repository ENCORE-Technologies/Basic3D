To run in local test:
Host the public_html folder in any http web hosting such as > python -m SimpleHTTPServer
Run http://localhost:8000/index.debug.html?ic=chair&sb=t&fm=f in Google Chrome Browser or
http://localhost:8000/index.debug.html?ic=ottoman&sb=t&fm=f for viewing another model

Sample models are stored in public_html/assets/models

For minified build you will need closure compiler. Goto public_html/build and run deploy.sh
